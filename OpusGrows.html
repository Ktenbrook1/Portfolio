Opus Grows was a very fun project to build. I made this before I even joined a boot camp. I was imitating the website Opus Grows just for fun to test my styling abilities. While it lacks the fancy databases and authentication it shows the attention to detail that I possess when given the time to invest in a project. It has hover features that allow for a better customer experience as well as a mobile design that doesn't break for customers on phones. All the CSS was written from scratch and there are no commits on GitHub because I never really dove into bootstrap/git before my boot camp. I did not follow any tutorial to build this.
