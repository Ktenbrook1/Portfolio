MovieLibrary is special because, on a team of two other individuals and I, we were able to build a REST API from scratch. I got to see the behind the scenes of what REST APIs are made from. After the construction of the API and testing through postman, we started to focus on the client-side of the application. We used jQuery, JSON, and AJAX to read the data from the database running on our local computers. I was able to find a source on WW3 schools that used jQuery to search through a database based on the titles or genre in live time. I even remade this exact project but using Node instead.
