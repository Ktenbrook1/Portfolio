CustomList is my take on Microsoft's version of their List class that allows for collections of multiple data types and objects. These objects/data types will expand as you add to them compared to arrays where you are required to initialize the amount of any certain data type you plan to track. I go into more detail on my Readme in GitHub click HERE to open that in a new window if you're interested.
